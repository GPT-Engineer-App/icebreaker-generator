# icebreaker-generator

A mobile-first application, designed to generate a single question that helps people get to know each other more. It will have several variables that the user can modify to change the next question, including Length: for shorter or deeper conversations, Familiarity: for strangers or close friends, and Nature: for casual or professional tones.

## Setup

```sh

cd icebreaker-generator

```


This will run a dev server with auto reloading and an instant preview.
